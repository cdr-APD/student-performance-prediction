# Student Performance Prediction using Machine Learning

This project predicts student math scores using machine learning models.  
The goal is to analyze student-related features and build models that can estimate performance.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models Implemented

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Project Workflow

1. Data loading using Pandas
2. Data exploration and visualization
3. Encoding categorical features
4. Splitting data into training and testing sets
5. Training multiple machine learning models
6. Evaluating models using R² Score and Mean Absolute Error
7. Comparing model performance with visualization
